# Sales Performance Excel Dashboard

## Overview
Interactive Excel dashboard built using Pivot Tables, Charts, Macros and Form Controls (Check Boxes).

## Features
- City-wise navigation
- Automated macros
- Interactive checkboxes
- KPI charts

## Tools Used
- Microsoft Excel
- VBA Macros

## How To Use
1. Download the file
2. Open in Excel
3. Enable Macros
4. Use buttons and checkboxes
